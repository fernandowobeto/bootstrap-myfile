jQuery MyFile Bootstrap

Estilização e eventos para o elemento file do HTML.

Download agora

Descrição

O jQuery MyFile Bootstrap é um plugin desenvolvido utilizando o jQuery com a estilização do Bootstrap, fornecendo uma maneira para deixar seus campos file mais atrativos e no padrão visual do seu projeto utilizando o Bootstrap.

Instalação

Inclua o arquivo bootstrap-myfile.js em seu código html.

<script src="js/bootstrap-myfile.js"> </script>

Dependências

jQuery

Bootstrap

Utilização

Uso básico:

$(":file").myfile();

Uso com opções:

$(':file').myfile({
    buttonText: 'Selecione',
    icon: false
});
