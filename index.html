jQuery MyFile Bootstrap

Estilização e eventos para o elemento file do HTML.

Download (3.1kb)

Descrição

O jQuery MyFile Bootstrap é um plugin desenvolvido utilizando o jQuery com a estilização usando as classes disponíveis no Twitter Bootstrap, fornecendo uma maneira para deixar seus campos file mais atrativos e no padrão visual do seu projeto utilizando o Bootstrap.

Instalação

Inclua o arquivo bootstrap-myfile.js em seu código html.

<script src="js/bootstrap-myfile.js"> </script>

Dependências

jQuery

Bootstrap

Utilização

Uso básico:

$(":file").myfile();

Uso com opções:

$(':file').myfile({
    buttonText: 'Selecione',
    icon: false
});

Opções

buttonText string Texto do botão
textField boolean Mostra ou não o campo text
icon boolean Mostra ou não o ícone no botão
classButton string Classe Bootstrap para o botão
classText string Classe Bootstrap para o campo text
classIcon string Classe Bootstrap para o ícone

Métodos

enable Habilita o botão para selecionar um arquivo
disable Desabilita o botão para selecionar um arquivo
clear Limpa o campo
